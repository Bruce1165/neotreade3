# Archive Docs

## 目的

本目录只存放已退出活跃路径的历史材料。

这些材料保留的目的是：

- 留痕
- 对照
- 审计

这些材料不承担当前正式口径职责。

## 使用规则

- archive 中材料必须被视为 `reference` 或 `archived`
- 如果某份材料仍是当前正式口径，则不应放在本目录
- 当活跃文档被新文档替代且不再承担当前职责时，应迁入本目录

## 当前子目录

- `debug/`
  - 历史调试记录
- `handover/`
  - 已退出活跃续接路径的历史交接材料

## 退出条件

本目录本身不删除，但目录内文件如果被确认：

- 无历史审计价值
- 无回退参考价值
- 无复盘价值

则应在单独清理动作中删除，而不是无限保留。
