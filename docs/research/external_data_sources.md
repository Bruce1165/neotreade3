## 三、NeoTrade3 集成建议

### a) 财务数据 (PE/PB/ROE) -- 当前覆盖率低

**推荐: 腾讯财经 + mootdx 季报快照 + 新浪财报三表**
- 腾讯财经: PE(TTM)/PB/市值，HTTP直连，零成本，与现有腾讯行情源复用
- mootdx 季报: 37字段含 ROE/EPS/净利润，Python库调用简单
- 新浪财报三表: 补充完整财务报表数据，HTTP直连
- 同花顺一致预期: 补充机构一致预期EPS，用于前向PE计算

### b) 政策/新闻数据 -- 当前缺失

**推荐: 财联社快讯 + 东财个股新闻 + 巨潮公告**
- 财联社: 分钟级电报，政策发布第一手来源，时效性最强
- 东财新闻: 个股维度新闻流，覆盖面广
- 巨潮公告: 官方公告源(财报/重组/减持等)，权威性最高
- 同花顺热点: 题材归因reason tags(人工标注)，可用于题材/政策热点追踪

### c) 日线行情备用源 -- 当前依赖腾讯

**推荐: mootdx 作为主备选 + 百度K线作为次备选**
- mootdx: TCP直连通达信，K线+盘口+逐笔，封IP风险极低，稳定性最佳
- 百度K线: HTTP接口，含MA5/10/20，轻量级备选
- 注意: mootdx需国内IP，海外环境需代理

## 四、接入优先级建议

1. **P0 (立即接入):** 腾讯财经PE/PB、mootdx季报(ROE/EPS)、财联社快讯
2. **P1 (短期接入):** 新浪财报三表、巨潮公告、同花顺一致预期
3. **P2 (中期接入):** 东财资金流向、龙虎榜、同花顺热点题材归因
4. **P3 (按需接入):** iwencai语义搜索(需Key)、融资融券、大宗交易
