# Legacy Lowfreq Engines

This directory stores historical lowfreq engine files that are kept for reference only.

These files are:

- readable
- non-authoritative
- excluded from active calibration
- excluded from official validation and current tuning work

The active lowfreq engine is:

- `../../lowfreq_engine_v16_advanced.py`

If you are tuning the current lowfreq model, do not edit files in this directory unless the task is explicitly archival or comparative research.
